## Icarus Sentinel (Server Manager)

A lightweight, Python-based desktop application designed to simplify the deployment and maintenance of a dedicated Icarus server. The goal is to offload game logic from the client to a dedicated process to improve FPS and stability, while automating weekly updates, save-game security, and memory optimization.

### Key Features

- **Safe Launch**: Pre-flight RAM checks and optional auto-update on launch ensure the server is always up-to-date with sufficient system resources
- **Smart Backup**: Automated, timestamped ZIP backups of world saves every 30 minutes and upon shutdown, with configurable retention policy to manage disk space
- **Save Sync**: Bidirectional synchronization of world saves between local single-player and the dedicated server using a "Keep Newest" conflict policy
- **Player Tracking**: Real-time monitoring of connected players, playtime, and server metrics via the Valve A2S protocol with integrated Windows desktop notifications
- **Automated CI/CD**: GitHub Actions pipeline for automated Windows executable builds and semantic versioning

### Tech Stack

Written in Python 3.11+ with PySide6 providing a high-fidelity industrial dashboard UI. System monitoring uses psutil, server discovery uses the python-a2s library (Valve A2S protocol), and native Windows notifications are powered by winotify.

### Architecture

The application follows a modular Controller-View pattern — business logic is fully decoupled from UI components. No single source file exceeds 500 lines. The core workflows (Safe Launch, Smart Backup, Save Sync, Player Tracking) operate as independent modules orchestrated by a central controller, making the system maintainable and extensible.
