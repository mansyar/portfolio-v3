## Icarus Server Manager

A lightweight, self-hosted server management dashboard that brings together system monitoring, Docker container management, and deployment orchestration in a clean, accessible web interface.

### Key Features

- **Real-time Monitoring**: Live system metrics (CPU, memory, disk, network) with historical graphing
- **Docker Management**: Start, stop, restart, and view logs for containers directly from the dashboard
- **Deployment Orchestration**: One-click deployment from GitHub webhooks with rollback support
- **User Management**: Role-based access control for team environments
- **Alert System**: Configurable thresholds with email and webhook notifications

### Tech Stack

Built with TypeScript across the stack — Node.js/Express backend, PostgreSQL for metrics storage, Redis for caching and real-time pub/sub, and Docker for containerized deployment.

### Architecture

The application follows a modular monolith pattern, with clear separation between the API layer, service layer, and data access layer. WebSocket connections provide real-time metric streaming to the dashboard frontend.
