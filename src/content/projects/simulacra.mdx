## Simulacra — Autonomous AI Ant Farm

Simulacra is an autonomous AI "Ant Farm" — a persistent virtual world where AI agents live, work, and socialize. Built with TanStack Start, PixiJS, and Convex. Currently in active development.

### Core Vision

- **Real-time Synchronization**: Persistent world state synced across all clients
- **Autonomous Agents**: AI agents with memory, goals, and social relationships using Groq (Llama 3)
- **16-bit Aesthetic**: Cozy isometric pixel-art interface inspired by Stardew Valley
- **God Mode**: Master controls for weather, time, and world resets

### Architecture

The project follows a clear separation between game rendering and UI logic:

```
src/
├── components/game/     # PixiJS rendering (Grid, Camera, Sprites)
├── components/ui/       # React UI (Header, AdminPanel, ThoughtStream)
├── convex/functions/    # Backend simulation loop, AI interactions, memory
└── docs/               # Technical specifications and architecture diagrams
```

### Tech Stack

| Layer             | Technology                          |
| ----------------- | ----------------------------------- |
| **Frontend**      | TanStack Start (React + TypeScript) |
| **Rendering**     | PixiJS v8 (GPU-accelerated 2D)      |
| **Backend/DB**    | Convex (Real-time sync + Vector Search) |
| **AI**            | Groq API (llama-3.1-8b-instant)    |
| **Styling**       | Tailwind CSS + Framer Motion        |

### Development Setup

```bash
# Install dependencies
pnpm install

# Configure Convex credentials in .env.local
# VITE_CONVEX_URL=your_convex_url

# Run development server
pnpm dev

# Start Convex backend
npx convex dev
```

### Current State (WIP)

The project is in active development with the core rendering pipeline and Convex backend integration being the primary focus areas. The PixiJS isometric grid rendering and agent simulation loop are the next milestones.
